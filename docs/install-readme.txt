JHakonen's mod for integrating TeamSpeak into World of Tanks.

Installation:
 - Extract the archive to your World of Tanks's mods folder, e.g:
     C:\Games\World_of_Tanks\<version>\mods
 - Download and install TeamSpeak plugin (optional, required for 3D audio);
     https://www.myteamspeak.com/addons/01a0f828-894c-45b7-a852-937b47ceb1ed

For more information, see:
  http://forum.worldoftanks.eu/index.php?/topic/433614-/

For source code and bug reports see:
  https://github.com/jhakonen/wot-teamspeak-mod
  https://github.com/jhakonen/wot-teamspeak-plugin
